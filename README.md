# Universal Window Mount for Through-Wall ERVs

This repository contains the design assets for a repeatable, secure, and insulated mounting system that allows commercially available through-wall Energy Recovery Ventilators (ERVs) to be installed in residential windows without permanent wall modifications.

## Project Goal
Standard through-wall ERVs (e.g., Lunos, Blauberg, Vents) typically require a 100mm or 160mm hole drilled through a masonry or stud wall. This project provides a "Window Adapter Kit" that:
1.  Holds the ERV tube securely in a sliding window (vertical or horizontal).
2.  Provides professional-grade thermal insulation and air sealing.
3.  Manages the cantilevered weight of the internal and external components.
4.  Remains easily removable for seasonal use or for renters.

## Directory Structure
- `/parts`: Parametric OpenSCAD design and 3D-printable assets.
- `requirements.md`: Technical constraints and target window dimensions.
- `assembly_guide.md`: Instructions for assembly and sealing.
- `pioneer_optimization.md`: Specific tweaks for the Pioneer ECOasis 50.
- `vents_optimization.md`: Specific tweaks for the Vents-US TwinFresh Comfo.
- `track_profiles.md`: Guide to different window track types.
